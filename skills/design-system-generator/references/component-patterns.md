# Component Patterns Reference / 组件样式模式参考

This document defines the CSS patterns for each UI component generated by the Design System Generator.

## General Rules / 通用规则

1. **Token-only values**: NEVER hardcode any color, font-size, spacing, radius, or shadow value. Always use `var(--token-name)`.
2. **BEM-ish naming**: Use flat class names with `--` for variants: `.btn`, `.btn--primary`, `.btn--sm`.
3. **State styling**: Use pseudo-classes `:hover`, `:focus-visible`, `:active`, `:disabled`, `[aria-invalid]`.
4. **Transitions**: All interactive state changes use `transition: var(--transition-fast)` or `var(--transition-normal)`.
5. **Focus visible**: All interactive elements MUST have a visible `outline` on `:focus-visible`.
6. **Dark mode automatic**: Components use semantic tokens that auto-switch via `light-dark()`. No extra dark mode CSS needed per component.

## Button / 按钮 `.btn`

```css
.btn {
  display: inline-flex;
  align-items: center;
  justify-content: center;
  gap: var(--spacing-2);
  padding: var(--spacing-2.5) var(--spacing-5);
  font-family: var(--font-family-sans);
  font-size: var(--font-size-sm);
  font-weight: var(--font-weight-medium);
  line-height: var(--line-height-none);
  letter-spacing: var(--letter-spacing-wide);
  border-radius: var(--radius-md);
  border: 1px solid transparent;
  cursor: pointer;
  transition: all var(--transition-fast);
  outline: none;
}

.btn:focus-visible {
  outline: 2px solid var(--color-primary-500);
  outline-offset: 2px;
}

.btn:active {
  transform: scale(0.97);
}

.btn:disabled {
  opacity: 0.5;
  cursor: not-allowed;
  transform: none;
}
```

### Variants:

```css
.btn--primary {
  background: var(--color-primary-600);
  color: var(--color-on-primary);
}
.btn--primary:hover {
  background: var(--color-primary-700);
}

.btn--secondary {
  background: var(--color-bg-secondary);
  color: var(--color-text-primary);
  border-color: var(--color-border);
}
.btn--secondary:hover {
  background: var(--color-bg-tertiary);
}

.btn--ghost {
  background: transparent;
  color: var(--color-primary-600);
}
.btn--ghost:hover {
  background: var(--color-primary-50);
}

.btn--danger {
  background: var(--color-error);
  color: var(--color-error-contrast);
}
.btn--danger:hover {
  background: var(--color-error-dark);
}
```

### Sizes:

```css
.btn--sm {
  padding: var(--spacing-1.5) var(--spacing-3);
  font-size: var(--font-size-xs);
}
.btn--lg {
  padding: var(--spacing-3) var(--spacing-6);
  font-size: var(--font-size-base);
}
```

## Input / 输入框 `.input`

```css
.input {
  display: block;
  width: 100%;
  padding: var(--spacing-2.5) var(--spacing-3);
  font-family: var(--font-family-sans);
  font-size: var(--font-size-sm);
  line-height: var(--line-height-normal);
  color: var(--color-text-primary);
  background: var(--color-bg-primary);
  border: 1px solid var(--color-border);
  border-radius: var(--radius-md);
  outline: none;
  transition: all var(--transition-fast);
}

.input::placeholder {
  color: var(--color-text-muted);
}

.input:focus {
  border-color: var(--color-primary-500);
  box-shadow: 0 0 0 3px var(--color-primary-100);
}

.input:disabled {
  opacity: 0.5;
  cursor: not-allowed;
  background: var(--color-bg-secondary);
}

.input--error,
.input[aria-invalid="true"] {
  border-color: var(--color-error);
}

.input--error:focus,
.input[aria-invalid="true"]:focus {
  box-shadow: 0 0 0 3px var(--color-error-light);
}
```

### Textarea:

```css
.textarea {
  /* Inherits all .input styles */
  min-height: 6rem;
  resize: vertical;
}
```
